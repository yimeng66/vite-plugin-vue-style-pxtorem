# vite-plugin-vue-style-pxtorem

一个vite插件转换vue模板内联样式px到rem

## Install

```bash
pnpm install vite-plugin-vue-style-pxtorem --save-dev
```

## options

Type: `Object | Null`  
Default:
```js
{
    rootValue: 16,
    unitPrecision: 5,
    minPixelValue: 0,
    exclude: []
}
```
